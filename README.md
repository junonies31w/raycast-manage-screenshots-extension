# Screenshot Manager

Manage your screenshots directly from Raycast.

## Features

- **View Customization**: Toggle between List and Grid views.
- **Grid Control**: Customize the number of grid columns (1-8).
- **Flexible Storage**: Set your custom screenshots folder in preferences.
- **Quick Actions**: Open, Copy to Clipboard, Show in Explorer, and Delete.

## Shortcuts

- `Ctrl + R` : Reload
- `Ctrl + Enter` : Open With
- `Ctrl + C` : Copy to Clipboard
- `Ctrl + X` : Delete File
- `Ctrl + V` : Switch View (List/Grid)
- `Ctrl + G` : Set Grid Columns
